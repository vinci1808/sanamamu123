Program Specifications =
1) Accept a Location from the User.
2) Access the VSAM KSDS Employee Master on the Loc AIX to Display All the
   Employees belonging to that Location.
===========================================
E1 L1
E2 L2
E3 L1
E4 L2
E5 L1
E6 L1
+++++
1)  SELECT EMPLOYEE-MASTER ASSIGN TO EMPMAST
    ORGANIZATION IS INDEXED
    ACCESS MODE  IS SEQUENTIAL
    RECORD KEY   IS EMP-ID
    Read Sequentially
    READ EMPLOYEE-MASTER
	AT END
=============================================
2)  SELECT EMPLOYEE-MASTER ASSIGN TO EMPMAST
    ORGANIZATION IS INDEXED
    ACCESS MODE  IS RANDOM
    RECORD KEY   IS EMP-ID
    Read Random
    READ EMPLOYEE-MASTER
	INVALID KEY
=============================================
3)  SELECT EMPLOYEE-MASTER ASSIGN TO EMPMAST
    ORGANIZATION IS INDEXED
    ACCESS MODE  IS DYNAMIC
    RECORD KEY   IS EMP-ID
    Read Random
    READ EMPLOYEE-MASTER
	INVALID KEY
    Read Sequentially from Beginning of the File
    READ EMPLOYEE-MASTER
	AT END
    To Read Sequentially from a specific Point
    MOVE WS-START TO EMP-ID
    START EMPLOYEE-MASTER
	KEY EQUAL TO EMP-ID
	  INVALID KEY
	     DISPLAY 'START NOT SUCCESSFUL'
	  NOT INVALID KEY
	     DISPLAY 'START WAS SUCCESSFUL'
    END-START
    READ EMPLOYEE-MASTER NEXT
	AT END
========================================================
    SELECT EMPLOYEE-MASTER ASSIGN TO EMPMAST
    ORGANIZATION IS INDEXED
    ACCESS MODE  IS SEQUENTIAL
    RECORD KEY   IS EMP-ID
    ALTERNATE RECORD KEY IS EMP-LOC WITH DUPLICATES
PROCEDURE DIVISION.
    ACCEPT WS-LOC
    MOVE WS-LOC TO EMP-LOC
    START EMPLOYEE-MASTER
	KEY = EMP-LOC
	   INVALID KEY
	     MOVE 'N' TO WS-START-FLAG
	   NOT INVALID KEY
	     MOVE 'Y' TO WS-START-FLAG
    END-START.
    IF START-SUCCESS
	PERFORM PRINT-LOC-PARA
    END-IF
PRINT-LOC-PARA.
    READ EMPLOYEE-MASTER NEXT RECORD
	AT END
	   MOVE 'Y' TO WS-FILE-FLAG
	NOT AT END
	   DISPLAY 'EMP ', MASTER-RECORD
    END-READ
