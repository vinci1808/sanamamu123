MAIN PARA
     PERFORM INIT PARA
     PERFORM PROCESS PARA UNTIL END OF FILE
     PERFORM END PARA,
INIT PARA
     OPEN INPUT FILE
     OPEN INPUT-OUTPUT FILE
     OPEN REPORT FILE
     DISPLAY IN STATUS
     DISPLAY I/O STATUS
     DISPLAY REP STATUS.
     PERFORM INPFILE READ PARA
PROCESS PARA
IF VALID CODE
     PERFORM I/O READ PARA
ELSE
     DISPLAY INVALID CODE
END IF.     

INPFILE READ PARA.
     READ INPUT FILE
     AT END
         MOVE Y TO INP FLAG
     NOT AT END
         MOVE N TO INP FLAG
      
I/O READ PARA.
     READ INPUT-OUTPUT FILE
     EVALUATE TRUE
           WHEN EMP-ADD
               PERFORM ADD-PARA
           WHEN EMP-UPD
               PERFORM UPDATE-PARA
           WHEN EMP-DEL
               PERFORM DELETE-PARA
        END EVALUATE.    
ADD-PARA
IF EMP FOUND

UPDATE-PARA
IF EMP FOUND

DELETE-PARA
IF EMP FOUND

END-PARA
